This plugin is going to make your move from CubeCart WooCommerce swift and easy.

== Description ==
Сart2Cart: CubeCart to WooCommerce Migration Plugin gives an opportunity to set a connection between your stores and provide a troubleless data move just in a couple of hours. Even a newbie can perform this switch with no efforts and get accurate migration results. Pay attention that CubeCart store is not going to be downtimed during the move. 

= Features of Automated Migration with Cart2Cart =
- **Simple** - perform a few steps following Migration Wizard instructions.   
- **Fast** - you need only a couple of hours to get a brand new WooCommerce store.
- **Free Demo** - a chance to try how service works in 10-30 minutes. 
- **Support** - you can get a hand of help using phone, live chat or email.

= The following entities can be moved from CubeCart to WooCommerce =
* Products, product images, product attributes;
* Customers, customer billing address;
* Orders, order statuses, order products;
* Currencies;
* Manufacturers;
* Taxes.

*Supported CubeCart versions:* 3.x, 4.x, 5.x.x. 
*Supported WooCommerce versions:* 1.4.x, 1.5.x, 1.6.x, 2.0.x (new software versions are constantly being added).

**Please note.** *This plugin gives a chance to transfer only data, so that your theme is not going to be moved to WooCommerce.*

Cart2Cart: CubeCart to WooCommerce Migration helps to install a Connection bridge for your both e-stores. It will provide an accurate data interaction with setting an appropriate connection between them. When it is done, there will be performed a redirection to Cart2Cart’s website, where you will have a possibility to run Full migration. 

= Steps to take before migration =
* Install WooCommerce and afterwards check whether both stores can be accessed via web. 
* Make sure you have CubeCart FTP details to perform Connection bridge installation. 


==Installation==
1. Download the plugin archieved file.
1. Unzip plugin file to your PC hard drive.
1. Upload extracted file to wp-content/plugin directory.
1. Proceed to Admin -> Plugins, activate “Cart2Cart CubeCart o WooCommerce Migration”.
1. Provide CubeCart FTP credentials and download Connection bridge. 
1. WooCommerce Connection bridge is going to be completed automatically.
1. Start Migration and after a redirection to Cart2Cart official page you will be able to finish your migration. 

== Frequently Asked Questions ==
= The URLs on migrated products/categories are incorrect =
You may need to clear the CubeCart cache and rebuild the SEO Links for categories/products after migration.
Please log into your Magento admin section, go to System -> Cache Management
Clear caches and rebuild all sections.

= Will Cart2Cart migrate my attribute sets? =
Unfortunately, Cart2Cart provides a possibility to migrate only your attributes. Attribute sets will be created automatically right after migration is completed.

= Can Cart2Cart switch domain names? =
Cart2Cart doesn’t provide the service of domain switching. Check the whole list of Services.

= Can I delete my Source Store data after migration? =
We highly recommend not to delete any of your entries (products, customers, orders, etc.) at the Source Store after full migration is completed. Since if you later on decide to use our Recent Data Migration Service we can't guarantee proper data transfer results.

= Cannot detect default language =
The default language is not specified. Please specify the default language in your source store settings.
